    Formalizmo


    Σ = {a,b}
    Q = {A, B, C, D, E, F, G}
    QA ={F, G, E}
    Qi ={A}

    Marcamos las celdas que su par corresponda a un Qa y un Q

    Verificamos las celdas no marcadas con la condición:


    (A,B) ={ δ(A,a) = B, δ(B,a) = D} No cumple.
    (A,B) = { δ(A,b) = A, δ(B,b) = F} Cumple.

    Siguiente par:


    (A ,C ) = { δ( A,a) =B , δ( C,a) = A} Cumple.
    ( A,C ) = { δ( A,b) = A, δ( C,b) = C} Cumple.

    Siguiente par:


    (A,D) = { δ(A ,a) = B, δ(D,a) = C} No cumple.
    (A,D) = { δ(A,b) = A, δ(D,b) = E} Cumple.

    Siguiente par:


    ( B,C ) = { δ(B ,a) = D , δ( C,a) = A} Cumple.
    ( B,C ) = { δ(B,b) = F, δ( C,b) = C} Cumple.

    Siguiente par:


    ( B,D ) = { δ( B,a) = D, δ(D,a) = C} No cumple.
    (B,D) = { δ( B,b) = F, δ( D,b) = E} Cumple.

    Siguiente par:


    ( C,D ) = { δ( C,a) = A, δ(D,a) = C} Cumple.
    ( C,D ) = { δ( C,b) = C, δ( D,b) = E} Cumple.

    Siguiente par:


    (F,G ) = { δ(F,a) = G , δ(G,a) = F} No cumple.
    (F,G) = { δ(F,b) = D, δ(G,b) = G} Cumple.

    Ya que todos los pares están marcados podemos concluir que este AFD está en su estado mínimo.
