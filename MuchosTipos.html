    Gramaticas de muchos tipos

    G1

    G1 = {V, Σ, P, S}
    Donde:
         V = {S, A, T}
         Σ = {x, y}
         P = { S → A,
         A → xA | xT,
         T → y | A }
         S = {S}
    Figura de arbol 1

    Por ende, el lenguaje de la gramática 1 es:
    L1= {xy, xxy, xxxy, xxxxy, xxxxxy, …}

    Por comprensión
    L1 = {X^n y |n >0}

    G4


    𝐺4 = { 𝑉, Σ, 𝑃, 𝑆 }

    V = {S, A, B,C}

    G5


    𝐺4 = { 𝑉, Σ, 𝑃, 𝑆 }
    Donde:
    V = {S, A, T}
    Σ = {a, b }
    P = { S → AA | λ,
         A → aA | aT,
        T → bT | aA | bb}
    S = { S }
    Figura del árbol 5

    Como se puede observar en la figura anterior el lenguaje 5 es un conjunto de solución vacio, es decir, L5 = {Ø}. Entonces, la gramática 5 genera un lenguaje vacio, esto debido a que cada derivación en cualquiera de las ramas del árbol cuenta con un símbolo no terminal, por lo tanto no genera palabras. Por ende los arboles de derivación particular no existen palabras que se puedan validar debido a que la gramática es vacía.

    Gracias a los árboles de derivación anteriores podemos determinar que: {W1,W2,W3} ∉ ⊥ (𝐺4)
